## Question


#### You will implement three functions that are;

- _read_sudoku_ : This function reads a completed _9x9_ sudoku from user. Then returns the sudoku as 2D integer list.
- _check_horizontal_ : This function takes a 2d integer list called _sudoku_ . This function checks every row in the sudoku and detects the mistakes. If a number is not exist in the row or exist more than one that is a mistake. Finally prints the mistakes ascending order in desired format like in the sample test case.
- _check_vertical_ : This function takes a 2d integer list called _sudoku_ . This function checks every column in the sudoku and detects the mistakes. If a number is not exist in the column or exist more than one that is a mistake. Finally prints the mistakes ascending order in desired format like in the sample test case.
 
### Input Format

- _9x9_ numbers -> [1,9]

### Constraints

- Each item of the sudoku -> [1,9]

### Output Format

- Check the sample test cases.

Sample Input 
```
483921657
967345821
251876493
548132976
729564138
136798245
372689514
814253769
695417482
```
Sample Output

```
In row 9 mistakes are: 3 4
In column 7 mistakes are: 3 4
```
